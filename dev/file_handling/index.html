ECHOAnalysis.merge_tables — Method.
merge_tables(folder, dataroot, fltr; suffix=fltr)

Recurssively walk through a folder and merge tables inside folders named dataroot with a given filter fltr. The fltr is assumed to be the end of the filename and removed, but a different suffix may be provided.

Usage: mergetables("biobakery", "metaphlan2", "profile.tsv")
