Working with Metadata

In the previous notebook, we generated separate metadata files for different tables in the FilemakerPro database. Now, we'll get these into a more usable format for analysis.

We can only concatenate tables if they all have the same columns, so I'll add a sampleID to all of the other observations to match what's in samples. The fecal sample sampleIDs are build from the subjectID and timepoint, so I'll do the same for other observations

A some of the :value fields have quotes or newlines in the field, which screws up parsing later. For now I will just replace them.

for i in eachindex(allmeta[:value])
    v = allmeta[i, :value]
    ismissing(v) && continue
    if isa(v, AbstractString) && occursin(r"\n", v)
        v = replace(v, r"\n"=>"___")
        allmeta[i, :value] = v
    elseif isa(v, AbstractString) && occursin(r"\"", v)
        v = replace(v, r"\""=>"'")
        allmeta[i, :value] = v
    end
end
